# CreditWise Loan System

An end-to-end supervised machine learning pipeline to predict loan approval.

## 📌 Project Overview
- Built using **KNN**, **Logistic Regression**, and **Naive Bayes** classifiers.
- Designed to handle **binary classification** for loan approval prediction.
- Includes **Exploratory Data Analysis (EDA)**, **feature engineering**, and **model evaluation**.

## ⚙️ Features
- End-to-end ML workflow: data preprocessing → training → evaluation.
- Multiple algorithms compared for performance.
- Evaluation metrics: **Precision**, **Recall**, and **F1-score**.
